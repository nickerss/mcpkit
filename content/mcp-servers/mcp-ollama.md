# mcp-ollama

Ollama MCP 支持在本地运行各类开源 LLM，包括 Llama 3、Mistral、Qwen 等模型。所有推理在本地完成，数据不出网络，适合对数据隐私有严格要求的医疗、法律、金融等场景。
