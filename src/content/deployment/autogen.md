# AutoGen (Microsoft)

AutoGen 是微软研究院推出的多 Agent 对话框架，支持多个 LLM Agent 之间的协作，并可无缝引入人类反馈。适合企业级复杂 AI 应用开发，深度集成 Azure OpenAI。
