# AI 智能电商运营套件

让 AI Agent 自动管理你的电商店铺——从商品上架、订单处理、支付对账到营销自动化，一套工具全搞定。

## 解决的问题

电商运营充满重复性劳动：手动上架商品、处理买家咨询、对账支付流水、操作物流系统。每一个环节都在消耗团队宝贵的时间。AI E-Commerce Agent Kit 将这些环节全面自动化，让运营团队专注于真正的增长工作。

## 适用场景

- **商品管理**：AI 自动批量上架、编辑、上下架商品
- **客服自动化**：AI 实时回复买家问题（退换货政策、物流查询、尺寸建议）
- **订单处理**：AI 监控异常订单，自动触发退款或人工介入
- **支付对账**：Stripe 交易自动汇总，生成财务对账报表
- **营销自动化**：Klaviyo 邮件流程自动触发（弃购召回、生日优惠、新品通知）

## 行业数据

- 2026 年电商弃购率约 **70%**，AI 自动化可有效挽回（Shopify Blog）
- **78%** 的企业已在运营中集成 AI（Arcade 调研 2026）
- 40% 的电商企业正在标准化产品页面以适配 AI Agent（2026 数据）

- Stripe MCP 支持支付、客户、订阅、退款完整能力（Stripe Docs 2026）

## 工具组合

### Starter — 刚起步的独立站

核心工具组合，覆盖最基础的店铺管理和收款场景。

- `shopify-mcp` — Shopify GraphQL Admin API，商品 / 顾客 / 订单管理
- `shopify-storefront-mcp` — 面向买家的 AI 购物助手，商品搜索与加购
- `stripe-mcp` — Stripe 支付、订阅、退款、对账

### Pro — 成长期品牌

在 Starter 基础上增加营销自动化和智能选品。

- `shopify-mcp` — Shopify 商品管理
- `shopify-storefront-mcp` — AI 购物助手
- `stripe-mcp` — 支付处理
- `klaviyo-mcp` — 邮件 / SMS 营销自动化
- `shopify-product-recommendations-mcp` — AI 选品推荐

### Enterprise — 规模化运营

全链路覆盖，含 OMS 库存履约、Adyen 多支付网关、数据分析。

- `shopify-mcp` — Shopify 全功能管理
- `shopify-storefront-mcp` — AI 购物体验
- `stripe-mcp` — Stripe 支付
- `klaviyo-mcp` — 全渠道营销
- `adyen-mcp` — Adyen 企业支付
- `onestock-oms-mcp` — 库存统一管理 & 履约承诺
- `commercetools-mcp` — 企业级商务平台
- `shopify-analytics-mcp` — 数据分析与洞察
