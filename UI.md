## 1. 设计规范（Design System）

### 1.1 色彩系统（双主题）

| Token | 色值 | 用途 |
|-------|------|------|
| `--bg-primary` | `#0A0A0B` | 主背景 |
| `--bg-secondary` | `#16161A` | 次级背景（卡片/面板） |
| `--bg-tertiary` | `#18181B` | 悬浮/输入框背景 |
| `--bg-elevated` | `#1F1F23` | 浮层/下拉菜单背景 |
| `--border-subtle` | `#27272A` | 细分隔线 |
| `--border-default` | `#3F3F46` | 默认边框 |
| `--text-primary` | `#FAFAFA` | 主要文字 |
| `--text-secondary` | `#B8B8C0` | 次要文字 |
| `--text-tertiary` | `#71717A` | 辅助/占位文字 |
| `--accent` | `#8B5CF6` | 主强调色（紫色） |
| `--accent-hover` | `#A78BFA` | 悬浮态 accent |
| `--accent-muted` | `#8B5CF620` | accent 半透明背景 |
| `--success` | `#22C55E` | 成功/认证通过 |
| `--warning` | `#F59E0B` | 警告/待审核 |
| `--danger` | `#EF4444` | 错误/危险操作 |

#### 日间模式（Light Mode）— 新增浅色主题

| Token | 色值 | 用途 |
|-------|------|------|
| `--bg-primary` | `#FFFFFF` | 主背景 |
| `--bg-secondary` | `#F5F5F7` | 次级背景（卡片/面板） |
| `--bg-tertiary` | `#EBEBEF` | 悬浮/输入框背景 |
| `--bg-elevated` | `#FFFFFF` | 浮层/下拉菜单背景 |
| `--border-subtle` | `#E5E5EA` | 细分隔线 |
| `--border-default` | `#D1D1D6` | 默认边框 |
| `--text-primary` | `#1A1A1A` | 主要文字 |
| `--text-secondary` | `#6B6B6B` | 次要文字 |
| `--text-tertiary` | `#98989F` | 辅助/占位文字 |
| `--accent` | `#8B5CF6` | 主强调色（紫色，保持一致） |
| `--accent-hover` | `#7C3AED` | 悬浮态 accent |
| `--accent-muted` | `#8B5CF615` | accent 半透明背景 |
| `--success` | `#16A34A` | 成功/认证通过 |
| `--warning` | `#EA580C` | 警告/待审核 |
| `--danger` | `#DC2626` | 错误/危险操作 |
| `--card-shadow` | `0 2px 8px rgba(0,0,0,0.08)` | 卡片阴影（浅色模式专用） |

**日间模式设计要点：**
- 背景色以白色/浅灰为主，保证屏幕阅读舒适度
- 卡片使用白色背景 + 柔和阴影，与深色背景形成层次对比
- 文字对比度：主文字 `#1A1A1A` 与背景 `#FFFFFF` 对比度约 19:1，符合 WCAG AAA
- 次文字 `#6B6B6B` 与背景 `#FFFFFF` 对比度约 7:1，符合 WCAG AA
- 边框色适当加深（`#D1D1D6`）以在浅色背景下保持可见性
- 按钮 hover 态加深 accent 色而非变亮（与夜间模式相反）

#### 主题切换方式

- **切换按钮**：导航栏右上角，登录按钮左侧
- **图标**：🌙（月亮）表示当前夜间模式，点击切换为日间模式；☀️（太阳）表示当前日间模式，点击切换为夜间模式
- **切换动效**：主题色平滑过渡（`transition: background-color 300ms, color 300ms`）
- **持久化**：用户选择存储在 `localStorage`，下次访问自动应用上次选择的主题
- **默认主题**：未登录用户默认夜间模式；已登录用户跟随个人设置

### 1.2 字体系统

- **UI 字体**：`Inter`, -apple-system, BlinkMacSystemFont, sans-serif
- **代码字体**：`JetBrains Mono`, `Fira Code`, monospace
- **字号规范**：

| Token | Size | Line Height | 用途 |
|-------|------|-------------|------|
| `--text-xs` | 12px | 16px | 标签/角标 |
| `--text-sm` | 14px | 20px | 正文辅助 |
| `--text-base` | 16px | 24px | 正文 |
| `--text-lg` | 18px | 28px | 大正文/副标题 |
| `--text-xl` | 20px | 28px | 卡片标题 |
| `--text-2xl` | 24px | 32px | 区块标题 |
| `--text-3xl` | 30px | 36px | Hero 副标题 |
| `--text-4xl` | 36px | 40px | Hero 主标题 |
| `--text-5xl` | 48px | 48px | 营销大标题 |

### 1.3 间距系统

基于 4px 网格：`4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48 / 64 / 80 / 96 / 128`

### 1.4 圆角规范

| Token | Radius | 用途 |
|-------|--------|------|
| `--radius-sm` | 6px | 按钮/输入框 |
| `--radius-md` | 8px | 卡片 |
| `--radius-lg` | 12px | 大卡片 |
| `--radius-xl` | 16px | 模态框 |
| `--radius-full` | 9999px | 标签/徽章 |

### 1.5 阴影

```css
--shadow-sm: 0 1px 2px rgba(0,0,0,.4);
--shadow-md: 0 4px 12px rgba(0,0,0,.5);
--shadow-lg: 0 8px 32px rgba(0,0,0,.6);
--shadow-glow: 0 0 24px rgba(139,92,246,.25); /* accent glow */
```

### 1.6 动效规范

- 过渡时长：`150ms`（快速）、`200ms`（标准）、`300ms`（强调）
- 缓动函数：`cubic-bezier(0.16, 1, 0.3, 1)`（snappy）
- 悬浮动效：transform scale(1.02) + border-color 过渡
- 页面切换：淡入淡出 + 轻微上移（opacity 0→1, translateY 8px→0）

### 1.7 图标系统

- **图标库**：Lucide Icons（线性风格，stroke-width: 1.5）
- **图标尺寸**：`16px`（内联）、`20px`（按钮）、`24px`（导航）

## 2. 导航结构（Navigation）

### 2.1 全局顶部导航栏

```
[Logo] | [探索工具] [Kit专区] [开发者] [定价] | [🌙/☀️] [语言] [登录] [注册]
```

**主题切换按钮：**
- 位置：登录按钮左侧
- 样式：图标按钮（32px × 32px），圆角 `--radius-full`，背景 `--bg-secondary`
- 图标：🌙（夜间模式指示）或 ☀️（日间模式指示）
- 交互：点击切换主题，图标随主题变化

**语言切换按钮：**
- 位置：主题切换按钮左侧
- 样式：下拉选择框，当前语言缩写显示（`EN` 或 `中文`）
- 下拉选项：`EN` / `中文`
- 交互：点击展开下拉菜单，hover 高亮选项，选中后关闭菜单并切换语言
- 图标：地球仪 `🌐` + 当前语言缩写

- **高度**：64px
- **背景**：`--bg-primary` + 底部 1px `--border-subtle`
- **Logo**：左侧，MCPKIT 文字 logo（Inter Bold）+ 紫色闪电图标
- **导航项**：水平排列，字号 `--text-sm`，`--text-secondary`，hover 时变为 `--text-primary` + 下划线 accent 色
- **CTA 按钮**：右侧，语言切换下拉 + 主题切换按钮 + 「登录」（ghost 按钮）+ 「注册」（filled accent 按钮）
- **i18n 支持**：所有页面关键文字均支持中英文切换，包括：导航菜单项、按钮文案、提示文字、错误信息等
- **滚动行为**：向下滚动时导航栏背景变为半透明毛玻璃效果（backdrop-filter: blur）

### 2.2 移动端导航

- 汉堡菜单（`Menu` 图标）触发侧边抽屉
- 抽屉从右侧滑入，宽 280px，包含完整导航项和 CTA
